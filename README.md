  python3.10 及以上 的标准库

这个程序的使用方法(以下两种任选一个):
  1.python gui_show.py
  2.同目录下直接执行 gui_show.exe （要求该文件和 student_data.db ，z.txt 同一个目录下）

    程序有三个界面: 登录，考试，展示 多次考试的平均分数(这个界面数据有些不明所以)
